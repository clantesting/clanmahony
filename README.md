# Clan O'Mahony Family Website

Static rebuild of clanmahony.com for free hosting on GitHub Pages.

**All images and photos load directly from the Internet Archive — no uploading needed!**

## GitHub Pages Setup (~10 minutes)

1. Go to github.com, sign up free
2. Click + → New repository → name it `clanmahony` → Public → Create
3. Click "uploading an existing file" → drag all files from this zip → Commit changes
4. Go to Settings → Pages → Deploy from branch → main / (root) → Save
5. Site live at: https://YOUR-USERNAME.github.io/clanmahony/

## Keep clanmahony.com (optional)
- In GitHub Pages settings enter `clanmahony.com` as custom domain
- At your domain registrar add 4 A records pointing to:
  185.199.108.153 / 185.199.109.153 / 185.199.110.153 / 185.199.111.153
- Cancel HostGator → save ~$285/year, just pay ~$15/year for the domain

## How it works
All theme images and family photos load straight from archive.org (Internet Archive).
Zero manual uploading required. The site is fully self-contained.
